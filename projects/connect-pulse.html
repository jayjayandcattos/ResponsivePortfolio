ConnectPulse

Description

ConnectPulse is a social media dashboard that allows users to manage and schedule posts across multiple platforms. It provides analytics on post engagement, follower growth, and other relevant metrics. Users can connect their social media accounts for seamless management.

Tech Stack

  • Frontend: Vue.js for building a reactive and dynamic user interface.
  • Backend: Laravel PHP for server-side application logic.
  • Database: MySQL for storing user data and analytics.
  • API Integrations: OAuth for connecting with social media platforms (e.g., Facebook, Twitter, Instagram).
  • Task Scheduling: Laravel Task Scheduling for managing background jobs like post scheduling.
