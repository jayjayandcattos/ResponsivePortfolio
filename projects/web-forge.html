WebForge

Description

WebForge is a web-based project management tool to help teams collaborate on tasks, track project progress, and manage deadlines. The tool includes features such as task creation, assignment, progress tracking, and file sharing.

Tech Stack

  • Frontend: Angular for building a dynamic and responsive user interface.
  • Backend: Python with Django for handling server-side logic.
  • Database: PostgreSQL for storing project data.
  • Real-time Updates: WebSocket integration for instant collaboration and updates.
  • Authentication: OAuth 2.0 for secure user login and authorization.
